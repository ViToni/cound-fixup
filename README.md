# "cound" fixup project

Fix usage of `cound` all over GitHub one PR at a time.

The use of `cound` is very versatile, e.g:

| Usage        | Instead of   |
| ------------ | ------------ |
| `cound`      | `count`      |
| `cound`      | `could`      |
| `cound't`    | `couldn't`   |
| `coundn't`   | `couldn't`   |
